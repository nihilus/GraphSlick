Installation
=============

Python
-------
Download and install https://pypi.python.org/pypi/ordered-set/1.1
You may need to install the setuptools first


IDA
--------

Unzip the GS package in %IDADIR%. It will extract:

plugins\GraphSlick.plw
and
plugins\GraphSlick\*.py <- python support files



Usage
========

Just go to a big function and press Ctrl-4
